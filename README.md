# currency-converter
Convert one currency to another while offline or online.

https://dikaeinstein.github.io/currency-converter

# Installation

Clone repo to your local machine:

```git
git clone https://github.com/Dikaeinstein/currency-converter.git
```

**Install dependencies and run locally**<br/>
*Note>> Install yarn if not already installed on local machine: 'npm i -g yarn'*

Then run:

```yarn
yarn install
```

Now start the app:

```yarn
yarn start
```
